La etiqueta <body> en HTML se utiliza para definir el cuerpo del documento. Todo el contenido visible de una página web, como texto, imágenes, enlaces y otros elementos, debe estar contenido dentro de esta etiqueta. El navegador interpreta y muestra todo lo que está dentro de <body> al usuario final. Es una de las etiquetas principales de un documento HTML, junto con <html> y <head>. Mientras que <head> almacena información invisible para el usuario (como el título o metadatos), es donde se coloca la información y elementos interactivos que forman la página web.
